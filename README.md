## nextjs-landing-page

This is a template page used for sharing links for personal use.

To set this up for your website, simply edit the following files:

```
// Profile Configuration
config/about.ts

// Page Configuration (Modify headerImage, profileImage)
config/page.ts

// Links Configuration
config/links.ts
```
